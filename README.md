📊 Business Insights Dashboard with AI-Generated Analytics
🔎 Project Overview

This project presents an end-to-end business analytics solution combining Python, Power BI, and AI-based insight generation to analyze sales performance, profitability, and customer behavior.

The objective is to transform raw transactional data into actionable business insights through data analysis, visualization, and automated insight generation.

🧰 Tools & Technologies

Python (Pandas, Data Analysis)

Power BI (Interactive Dashboard)

Git & GitHub (Version Control)

Data Visualization

AI-Driven Insight Automation

📈 Key Business Insights

Technology category generates the highest profit.

Furniture category shows weaker profitability despite strong sales.

West region contributes the largest share of overall profit.

Certain sub-categories consistently produce losses.

Interactive slicers for dynamic filtering
